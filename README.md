# fed4iot-cbpf-pana

## Create directory
```bash
$cd ~/fed4iot-cbpf-pana/src/FaceDetectionAndRecognition/
$mkdir image/find_image
$mkdir image/temp_image
$mkdir cache
```

## Usage  
```bash
$cd ~/fed4iot-cbpf-pana/src/FaceDetectionAndRecognition  
$python3 restapi_main.py
```
